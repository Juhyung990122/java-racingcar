## Test Naming Convention

- Method : 실제 메소드명 기반
    - {메소드이름}
    - {메소드이름}_exceptionBy{예외발생이유}
- DisplayName : 문서처럼 자세히 기술 ( 명확하고 간료하게 )
    - ex) 홀수를 받아 ~~반환한다.
    - ex) 홀수를 받아 3으로 나눈 나머지를 반환한다.
